暂存区

暂存区跟踪管理的是文件内容的改变，而不是文件的改变。

撤销暂存区 暂存区的实体放在仓库根目录下的隐藏文件.git中

git reset HEAD 用它可以撤销add操作，工作区不变。

对reset命令需要注意，它除了上述功能外，还被用于版本回退。
